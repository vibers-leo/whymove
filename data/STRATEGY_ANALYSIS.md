## 1. 현재 상태 진단

### 개발 완성도
- **코드 규모**: Next.js 16 (canary) + React 19, Supabase 실시간 DB, Lightweight Charts
- **주요 기능 구현 상태**:
  - ✅ 실시간 대시보드 (암호화폐/금융 시세)
  - ✅ 드래그 가능한 위젯 레이아웃 (react-grid-layout)
  - ✅ 고급 차트 (TradingView Lightweight Charts)
  - ✅ 실시간 데이터 스트림 (Supabase 실시간)
  - ✅ 다크/라이트 테마 (next-themes)
  - ✅ 반응형 디자인
  - ⚠️ 실제 API 데이터 소스 (테스트용 목 데이터만 있음)
  - ⚠️ 사용자 인증 (구조 준비, 미구현)
  - ⚠️ 알림 시스템 (미구현)
  - ⚠️ 포트폴리오 추적 (미구현)

### 매출/사용자 현황
- **현재 상태**: 프로토타입/데모 단계
- **배포**: Vercel 배포 가능
- **사용자**: 미측정 (데모 전용)
- **추정 상태**: 기술 데모 수준, 실제 사용자 검증 필요

### 핵심 과제
1. **데이터 소스 부재**: 실제 암호화폐/금융 API 미연동 (목 데이터만 있음)
2. **비즈니스 모델 불명확**: 수익화 전략 미정 (구독? 광고? 기업용?)
3. **사용자 가치 불명확**: "왜 이 대시보드를 써야 하는가?" vs 기존 플랫폼 (업비트, 바이낸스, 트래딩뷰)
4. **인증 미완성**: 사용자 로그인/저장 기능 미구현

## 2. Startup Canvas (11섹션 요약)

| 항목 | 내용 |
|------|------|
| **Vision** | 암호화폐와 전통 금융을 통합하고, 개인화된 위젯으로 자신의 포트폴리오를 한눈에 모니터링하는 올인원 실시간 대시보드 |
| **Market Segments** | 암호화폐 트레이더, 주식/선물 투자자, Multi-Asset 투자가, 개인 금융 관리자 |
| **Value Proposition** | 업비트/바이낸스/주식 앱을 모두 열지 않고, 한 대시보드에서 모든 자산 모니터링 + 커스텀 위젯 |
| **Trade-offs** | 기능 풍부함 vs 단순성 / 실시간성 vs 비용(API) / 보안 vs 접근성 |
| **Key Metrics** | 월간 활성 사용자(MAU), 세션 시간, 위젯 커스텀 비율, 포트폴리오 추적 전환율 |
| **Growth** | Reddit/Discord 암호화폐 커뮤니티, 트레이딩 블로그/유튜버 협력, Product Hunt |
| **Capabilities** | Supabase 실시간 DB, Lightweight Charts (TradingView), React Grid Layout |
| **Can't/Won't** | 자동 매매(Bot) 기능 없음 / 금융 투자 조언 불가 / 법인 계정 없으므로 개인 사용자만 |
| **Cost Structure** | Vercel, Supabase, 암호화폐/금융 API 구독(CoinGecko, Alpha Vantage, etc.), 개발 인력 |
| **Revenue Streams** | 프리미엄 구독(고급 차트, 무제한 위젯), API 연동료, 기업용 대시보드 |

## 3. Beachhead 분석

**Primary Segment: "암호화폐 트레이더 (활동적인 단기 매매자)"**

| 기준 | 평가 | 근거 |
|------|------|------|
| **Burning Pain** | ⭐⭐⭐⭐ | 여러 앱/탭 관리 번거로움, 실시간성 중요, 포트폴리오 손실 추적 필요 |
| **Willingness To Pay** | ⭐⭐⭐⭐ | 고수익 트레이더, 월 10-30만원 도구 비용 소비 의향 높음 |
| **Winnability** | ⭐⭐⭐ | 경쟁 치열 (트래딩뷰, 업비트 프로), 차별화는 위젯 + 통합 |
| **Referral** | ⭐⭐⭐⭐ | 활동적인 커뮤니티, 좋은 도구는 빠르게 전파 |

## 4. Growth Loop

**적용 타입**: 바이럴 (커뮤니티) + 네트워크 효과 (포트폴리오 공유)

- **Loop 1 (Community Viral)**: 트레이더 커뮤니티(Reddit/Discord) → 추천 → 실시간성 발견 → 입소문
  - 현재 상태: 미구현 (제품 아직 데모 수준)
  - 임계점: 차별화된 기능 (위젯 + 통합) 증명 필요

- **Loop 2 (Network Effect)**: 포트폴리오 공유 → 친구들 비교/영감 → 신규 가입
  - 현재 상태: 포트폴리오 추적 미구현
  - 예상 효과: 공유 기능 있을 시 20-30% 전환율

- **Loop 3 (Data Moat)**: 사용자 데이터 축적 → 개인화 추천 → 체류시간 ↑

## 5. 가격 전략

### 현재 정책
- **미정** (데모 단계)

### 추천 가격 모델
**Freemium + 기업용 B2B**

| 플랜 | 가격 | 대상 | 기능 |
|------|------|------|------|
| **Free** | 무료 | 모든 트레이더 | 기본 데이터 (1시간 지연), 3개 위젯 커스텀 |
| **Pro** | 12,900원/월 | 활동적 트레이더 | 실시간 데이터, 무제한 위젯, 고급 차트, 포트폴리오 추적 |
| **Plus** | 29,900원/월 | 전문 트레이더 | Pro + 알림 시스템, 자동 리포팅, API 접근 |
| **Enterprise** | 협의 | 자산운용사/펀드 | 흰색 레이블(White Label), 팀 협업, 전용 지원 |

**수익 예상** (1,000 활성 사용자, 15% Pro 전환)
- Free: 850명 (수익 0)
- Pro: 140명 × 12,900 = 1,806만원/월
- Plus: 10명 × 29,900 = 299만원/월
- **월 수익**: 약 2,100만원

## 6. 8-Risk 진단

| 리스크 | 수준 | 설명 | 대응 |
|--------|------|------|------|
| **Value Risk** | ⭐⭐⭐ | "기존 앱들로 충분한데 왜 이걸 써야 해?" 명확한 가치 필요 | 위젯 커스터마이제이션이 핵심 차별화, 데모 강조 |
| **Usability Risk** | ⭐⭐ | UI/UX 현대적, 학습곡선 낮음 | 지속 개선, 튜토리얼 추가 |
| **Viability Risk** | ⭐⭐⭐⭐ | 데이터 API 비용 높음, 실제 수익화 어려울 수 있음 | CoinGecko 무료 API부터 시작, 구독료로 커버 계산 |
| **Feasibility Risk** | ⭐⭐⭐ | 실시간 데이터 스트림 구현 복잡, API 연동 필요 | CoinGecko(무료), Alpha Vantage(주식) 부터 시작 |
| **Ethics Risk** | ⭐ | 사기성 금융상품 광고 위험, 과도한 거래 조장 | "투자는 본인 판단" 명시, 책임감 있는 마케팅 |
| **GTM Risk** | ⭐⭐⭐ | 암호화폐 규제 강화, 신뢰도 낮음(Crypto Winter) | 전통 금융과 통합으로 신뢰도 ↑, 기업용 B2B 채널 개발 |
| **Strategy Risk** | ⭐⭐⭐⭐ | 경쟁 매우 치열, 기술 진입장벽 낮음 (API 활용만 하면 됨) | 위젯 + UX 차별화만 경쟁력, 빠른 시장 선점 필수 |
| **Team Risk** | ⭐⭐ | 개인 프로젝트, 규제 대응 어려움 | 암호화폐 규제 전문가 자문 필요 |

## 7. North Star Metric

**추천 NSM**: 월간 포트폴리오 추적 사용자 수
- 핵심 기능 실행 지표 (사용자가 정기적으로 자산 모니터링)
- 프리미엄 전환과 직결 (추적 사용자 = Pro 후보)
- 목표: 월 5,000명

**Input Metrics**:
1. **실시간 데이터 요청 수** → 접속 빈도/열정도
2. **위젯 커스텀 비율** → 개인화 수용도
3. **세션 시간** → 몰입도
4. **Pro 전환율** → 지불 의향 (KPI)

## 8. 3단계 로드맵

### Phase 1 (현재~1개월): MVP 완성 + 실제 데이터 연동
**목표**: 실시간 데이터 API 연동, 베타 사용자 500명
- [ ] CoinGecko API 연동 (암호화폐 실시간 시세)
- [ ] Alpha Vantage API 또는 한국 증권 API (주식 시세)
- [ ] 사용자 인증 시스템 (Supabase Auth)
- [ ] 포트폴리오 기본 추적 (매수가, 현재가, 손익)
- [ ] 베타 사용자 500명 모집 (Reddit, Discord 암호화폐 커뮤니티)

**성공 기준**:
- 실시간 데이터 지연 <1초
- 포트폴리오 추적 정확도 95% 이상
- 베타 사용자 500명 확보
- NPS 60점 이상

### Phase 2 (1~3개월): 기능 고도화 + 초기 수익화
**목표**: 활성 사용자 2,000명, Pro 전환율 10%, 월 수익 500만원
- [ ] 알림 시스템 (가격 변동, 손절/익절)
- [ ] 자동 리포팅 (일일/주간 포트폴리오 요약)
- [ ] 고급 차트 (기술적 분석 지표)
- [ ] 포트폴리오 공유 기능 (친구와 비교)
- [ ] 프리미엄 구독 시스템 (Pro/Plus)
- [ ] 마케팅 시작 (Reddit, Twitter, YouTube)

**성공 기준**:
- 활성 사용자 2,000명
- Pro 전환율 10% (200명)
- 월 수익 500만원 이상
- 세션 시간 평균 10분 이상

### Phase 3 (3~6개월): 플랫폼 확장 + 기업 B2B
**목표**: 활성 사용자 5,000명, 월 수익 2,000만원, 기업 고객 3곳
- [ ] 추가 자산 클래스 (원자재, 채권, 외환)
- [ ] AI 투자 조언 (ChatGPT 기반, "정보 제공용")
- [ ] 커뮤니티 포럼 (트레이딩 아이디어 공유)
- [ ] 기업용 White Label 대시보드 (자산운용사)
- [ ] 모바일 앱 출시 (React Native)
- [ ] 국제 확장 (영문 버전)

**성공 기준**:
- 활성 사용자 5,000명
- Pro+Plus 구독자 750명 (15% 전환)
- 월 수익 2,000만원 이상
- 기업 고객 3곳 (White Label)

## 핵심 한 줄 요약

> **"암호화폐와 전통 금융을 하나의 커스터마이제이션 가능한 실시간 대시보드에서 통합 관리하는, 트레이더의 신뢰받는 모니터링 플랫폼."**

### 전략 핵심 3가지
1. **차별화**: 위젯 기반 개인화 (vs 기존 앱은 고정 레이아웃)
2. **실시간성**: 1초 이내 데이터 업데이트 (vs 지연 데이터)
3. **통합**: 암호화폐 + 주식 + 선물을 한 화면에 (vs 앱 전환 필요)
