# WhyMove 기능 개발 로드맵

> **핵심 가치**: 선물 트레이더에게 "왜 움직이는지"를 가장 빠르게 알려주는 변동성 알림 서비스
>
> ⚠️ 우리는 방향(오를지/내릴지)을 예측하지 않습니다. 변동성 발생을 알릴 뿐입니다.

---

## 🎯 핵심 기능 (Core Features)

### 1. 소셜 미디어 실시간 추적 ⭐ 최우선

**목표**: X, 스레드, 트루스소셜에서 주요 인플루언서의 게시물을 실시간으로 수집

| 기능                 | 설명                                                                | 우선순위 | 상태    |
| -------------------- | ------------------------------------------------------------------- | -------- | ------- |
| 인플루언서 목록 관리 | 추적할 계정 리스트 관리 (트럼프, 머스크, 주요 크립토 인플루언서 등) | 높음     | ⏳ 예정 |
| X(Twitter) 크롤링    | Twitter/X API 또는 크롤링으로 실시간 트윗 수집                      | 높음     | ⏳ 예정 |
| 트루스소셜 크롤링    | 트럼프 게시물 실시간 수집                                           | 높음     | ⏳ 예정 |
| 스레드 크롤링        | 주요 인플루언서 스레드 게시물 수집                                  | 중간     | ⏳ 예정 |
| 자동 번역            | 영어 게시물 한국어 자동 번역                                        | 중간     | ⏳ 예정 |
| 중요도 태깅          | AI로 게시물의 시장 영향도 분석 (High/Medium/Low)                    | 중간     | ⏳ 예정 |

**기술 스택 옵션**:

- Twitter API (유료) 또는 Nitter 크롤링
- Puppeteer/Playwright 기반 스크래핑
- Supabase Edge Functions로 백그라운드 수집

---

### 2. 경제 이벤트 캘린더 ⭐ 최우선

**목표**: CPI, FOMC, 고용지표 등 변동성을 유발하는 경제 이벤트 일정 알림

| 기능             | 설명                                      | 우선순위 | 상태    |
| ---------------- | ----------------------------------------- | -------- | ------- |
| 경제지표 일정 DB | 주요 경제지표 발표 일정 데이터베이스      | 높음     | ✅ 완료 |
| D-day 알림       | 중요 이벤트 카운트다운 표시               | 높음     | ✅ 완료 |
| 정치 일정        | 트럼프 연설, 백악관 브리핑 등 정치 이벤트 | 중간     | ✅ 완료 |
| 캘린더 위젯      | 대시보드에 경제 캘린더 표시               | 중간     | ✅ 완료 |
| 구글 캘린더 연동 | 사용자 개인 캘린더와 동기화               | 낮음     | ⏳ 예정 |

**데이터 소스**:

- Investing.com Economic Calendar
- ForexFactory
- FedWatch Tool
